Documentation

Sample request


A simple GET request to the endpoint with querystring URL, as per example below: 


curl http://fetchtext-api.herokuapp.com/fetch/?url=http://www.polygon.com/2014/6/26/5842180/shovel-knight-review-pc-3ds-wii-u
